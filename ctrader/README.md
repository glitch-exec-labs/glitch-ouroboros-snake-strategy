# cTrader Track

This folder documents the next platform direction for Ouroboros Snake Strategy.

## Goal

Preserve the ensemble identity while replacing the MT5 execution layer with a cleaner cTrader-native architecture.

## Preserve

- Oracle coordination
- six-snake strategy roles
- shared risk policy
- normalized signal model

## Replace

- MT5 terminal coupling
- MT5-specific order and symbol handling
- platform-specific runtime assumptions
