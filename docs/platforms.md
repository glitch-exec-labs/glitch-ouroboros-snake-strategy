# Platforms

## MT5 Today

The included implementation is currently MT5-oriented.

## cTrader Next

The long-term platform target is a cleaner cTrader-native deployment with the same strategy identity and a more portable execution adapter.

## Broker Portability Direction

The broader Glitch ecosystem has already shown multiple broker paths over time.

The goal is to preserve:

- strategy logic
- Oracle behavior
- risk policy

while allowing broker and venue adapters to change.
