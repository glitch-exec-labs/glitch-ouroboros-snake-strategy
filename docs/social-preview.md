# Social Preview Brief

This brief defines the GitHub social preview image for Ouroboros Snake Strategy.

## Purpose

Ouroboros should read as the flagship system in the Glitch family.

The image should communicate:

- Oracle coordination
- six-snake ensemble structure
- recursive portfolio intelligence
- flagship identity

## Export Spec

- Size: `1280 x 640`
- Format: `PNG`
- Safe area: keep all important text inside the left `60%` of the canvas
- Style: premium, cinematic, technical

## Exact Text

- Title: `OUROBOROS SNAKE STRATEGY`
- Subtitle: `Flagship Ensemble`
- Tagline: `Oracle coordination with the six-snake execution stack.`
- Footer strip: `Flagship System in the Glitch Ecosystem`

## Layout

- Place the title in the upper-left quadrant
- Set the subtitle directly below the title
- Keep the tagline short and high-confidence
- Dedicate the right half to a circular flagship motif
- Keep the footer strip along the bottom edge

## Visual Direction

- Build an abstract ouroboros-inspired circular form around a glowing core
- Suggest six incoming branches or nodes feeding into the Oracle center
- Keep the image symbolic rather than illustrative
- Add subtle motion and recursion cues without clutter

## Palette

- Background: black and deep graphite
- Primary accent: emerald
- Secondary accent: electric blue
- Optional depth accent: restrained violet

## Typography

- Title: bold uppercase with flagship weight
- Subtitle: smaller uppercase with clean tracking
- Tagline: minimal sans-serif, short enough to survive small previews

## Avoid

- no fantasy snake art
- no literal reptile rendering
- no overloaded diagrams
- no more than one circular hero motif
- no noisy gradients that reduce text legibility
